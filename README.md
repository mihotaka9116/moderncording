Tohno Modern Retreat 

岩手県遠野市の古民家再生宿「Tohno Modern Retreat（遠野モダンリトリート）」の架空のコンセプトサイトです。
伝統的な「遠野物語」の世界観と、現代的なリトリート（静養）の快適さを融合させた「和モダン」なデザインと、実用的な予約システム・UIを実装しました。

✨ 実装のこだわりと技術的ポイント
1. ユーザー体験（UX）を支えるJavaScript
ライブラリに依存せず、素のJavaScriptで「使いやすさ」と「軽量動作」を両立させました。

インテリジェントな予約バリデーション: - input[type="date"] を制御し、過去の日付を選択不可に設定。

チェックイン日の変更に合わせてチェックアウトの最小日を動的に更新。入力ミスを未然に防ぐ「フェイルセーフ」な設計です。

Intersection Observerによる高パフォーマンス演出:

スクロールイベントではなく、ブラウザ標準のAPIを使用。要素が画面の20%に達した際、滑らかにクラスを付与してフェードインさせることで、CPU負荷を抑えつつ上質な閲覧体験を提供します。

シングルページ最適化ナビゲーション:

モバイル版において、ページ内リンクをクリックした際に自動でドロワーメニューを閉じる処理を実装。ユーザーの操作ステップを最小化しました。

2. 保守性の高いCSS設計
CSSカスタムプロパティ（変数）: - 色彩（メイン、アクセント、背景色）を変数で一括管理。ブランドカラーの変更に即座に対応可能な柔軟な設計です。

流体タイポグラフィ（Responsive Typography）: - clamp() 関数を活用し、デバイス幅に応じて文字サイズをシームレスに調整。どの画面サイズでも美しいジャンプ率（文字の強弱）を維持します。

洗練された和モダンデザイン: - 背景にわずかなベージュ（#fdfcf9）を採用し、純白よりも目に優しく、古民家の温かみを感じさせる配色を行いました。

3. モバイルファーストなレスポンシブ対応
FlexboxとCSS Gridを使い分け、複雑な「お部屋・プラン」カードや「フォトギャラリー」をデバイスごとに最適化。

モバイル環境（767px以下）では、予約フォームを縦並びに再レイアウトし、親指一本で操作しやすいUIを実現しています。

📂 主なファイル構成
index.html : セマンティックなマークアップによる構造化

style.css : 変数、共通パーツ、各デバイス用メディアクエリを整理して記述

script.js : 予約ロジック、メニュー制御、アニメーション監視を一括管理

🎨 デザインコンセプト
「物語の続きを、この場所で。」
遠野に流れる静寂と、薪ストーブや南部鉄器といった「手触り感のある上質さ」を視覚化。
余白を贅沢に使い、ユーザーがサイトを訪れた瞬間からリトリート（日常からの切り離し）が始まるような体験を目指しました。
